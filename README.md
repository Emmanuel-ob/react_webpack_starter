# React & Webpack Starter Pack

This package gives you everything you need to build React ES6 applications with webpack

### Version
1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser

```sh
$ npm start
```

### Build
Compile and build

```sh
$ npm run build
```
